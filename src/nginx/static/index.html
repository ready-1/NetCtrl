System Information

NetCtrl CMS with RBAC is a comprehensive content management system featuring role-based access control. The system is designed to operate in environments with periodic internet outages.

Key Features

  • Role-Based Access Control (RBAC)
  • Content Management
  • API-First Architecture
  • Resilient to Network Outages
  • Centralized Logging
  • Scalable Containerized Deployment

System Architecture

This system is built using a microservices architecture with the following components:

  • FastAPI Backend: Handling business logic and API endpoints
  • PostgreSQL Database: Storing application data
  • React Frontend: Providing the user interface
  • NGINX: Serving as reverse proxy and static file server
