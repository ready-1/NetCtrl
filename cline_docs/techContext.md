# Technical Context

## Technology Stack

### Backend
- Python 3.12
- Django 5.1.5
- Django REST Framework
- Django Channels
- Gunicorn with Uvicorn workers
- Poetry for dependency management

### Database
- PostgreSQL 15 (Production)
- SQLite (Development)
- Redis for caching and channels

### Frontend
- Django Templates
- HTMX for dynamic updates
- Bootstrap for styling
- Custom CSS/JS

### Infrastructure
- Docker Engine 24.0+
- Docker Compose
- Nginx Alpine
- SSL/TLS certificates

## Development Environment

### Requirements
- Python 3.12+
- Poetry
- Docker Engine 24.0+
- Git

### Local Setup
```bash
# Install dependencies
poetry install

# Activate environment
poetry shell

# Run development server
python manage.py runserver
```

### Development Tools
- Black for code formatting
- Flake8 for linting
- Pytest for testing
- Pre-commit hooks

## Production Environment

### System Requirements
- Debian 11+ or Ubuntu 22.04+
- 2GB RAM minimum
- 20GB disk space
- Docker Engine 24.0+
- Available ports: 80, 443

### Directory Structure
```
/opt/netctrl/
├── app/              # Application code
├── static/           # Static files
├── media/           # Media files
├── logs/            # Application logs
└── certs/           # SSL certificates
```

### Environment Variables

#### Database Configuration
```bash
POSTGRES_DB=netctrl
POSTGRES_USER=netctrl
POSTGRES_PASSWORD=<auto-generated>
POSTGRES_HOST=db
POSTGRES_PORT=5432
```

#### Django Settings
```bash
DJANGO_SETTINGS_MODULE=config.settings
DJANGO_SECRET_KEY=<auto-generated>
DJANGO_DEBUG=False
DJANGO_ALLOWED_HOSTS=<domains>
```

#### File Paths
```bash
STATIC_ROOT=/opt/static
MEDIA_ROOT=/opt/media
LOG_DIR=/opt/logs
```

#### Redis Configuration
```bash
REDIS_HOST=redis
REDIS_PORT=6379
```
